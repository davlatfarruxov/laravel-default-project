# MarketingPro — v3.0

> v3.0 = v2.0 marketing yadrosining sayqallangan, **ikki tilli (EN/UZ)**,
> demo ma'lumotlar bilan to'ldirilgan va **issiq Coral→Amber dizaynli** varianti.

---

## Bu versiyada nima bor

### 🎨 Dizayn
- To'liq yangi **Coral → Amber** issiq palitra (`resources/css/theme.css`).
- Dark = warm charcoal, Light = warm cream. Radius, soyalar, glow yangilangan.
- Analitika: Chart.js bilan line / 2× doughnut / grouped bar + CSS funnel.

### 🌐 Ikki tillilik (EN / UZ)
- `lang/en/*` va `lang/uz/*` — common, nav, auth, toast, dashboard, leads,
  campaigns, analytics, info.
- `SetLocale` middleware (standart: **uz**, fallback: **en**).
- `LocaleController` + `GET /locale/{locale}` route (`locale.switch`).
- Header'da til almashtirgich (UZ/EN), login sahifada ham.
- Barcha asosiy sahifalar `__()` orqali tarjima qilingan: sidebar, header,
  dashboard, login, leads, campaigns, analytics, resurs sahifalari.

### 👤 Oddiy foydalanuvchilar uchun
- Yangi **`user`** roli (faqat ko'rish: dashboard, leads, campaigns, analytics).
- 3 ta ochiq sahifa (barcha foydalanuvchilar): **Maslahatlar**, **Savol-javob**,
  **E'lonlar** (`InfoController` + `info.*` route + sidebar "Foydali" bo'limi).
- Dashboard'da oddiy foydalanuvchi uchun "Boshlash" yo'riqnoma paneli.

### 🌱 Seederlar (demo ma'lumot)
- `CampaignFactory`, `LeadFactory`.
- `UserSeeder` (manager + 3 oddiy foydalanuvchi), `CampaignSeeder` (12),
  `LeadSeeder` (60, kampaniya/xodimga bog'langan).
- `DatabaseSeeder`: RolePermission → User → Campaign → Lead.

## Ishga tushirish
```bash
cp .env.example .env      # APP_LOCALE=uz, fallback en
php artisan key:generate
php artisan migrate:fresh --seed
npm install && npm run build
php artisan serve
```
