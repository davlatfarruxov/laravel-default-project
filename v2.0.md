## Yangi modullar

### 1. 📇 Mijozlar (Leads / Contacts)
Potensial va mavjud mijozlar bazasi — marketing "voronka"sining asosi.

**Maydonlar:**
| Maydon | Tavsif |
|--------|--------|
| `name` | Mijoz ismi |
| `email` | Email manzil |
| `phone` | Telefon raqami |
| `company` | Kompaniya nomi |
| `source` | Manba (website, social, referral, ads, event, other) |
| `status` | Holat (new, contacted, qualified, won, lost) |
| `value` | Potensial qiymat (so'm) |
| `campaign_id` | Qaysi kampaniyadan kelgan (nullable) |
| `assigned_to` | Mas'ul foydalanuvchi (nullable) |
| `notes` | Izohlar |

**Imkoniyatlar:** qidiruv, holat bo'yicha filtr, saralash, sahifalash, CRUD.
**Ruxsatlar:** `leads.view`, `leads.create`, `leads.edit`, `leads.delete`.

---

### 2. 📢 Kampaniyalar (Campaigns)
Marketing kampaniyalarini rejalashtirish, byudjet va natijalarni kuzatish.

**Maydonlar:**
| Maydon | Tavsif |
|--------|--------|
| `name` | Kampaniya nomi |
| `channel` | Kanal (email, social, seo, ppc, sms, event) |
| `status` | Holat (draft, active, paused, completed) |
| `budget` | Ajratilgan byudjet |
| `spent` | Sarflangan summa |
| `start_date` | Boshlanish sanasi |
| `end_date` | Tugash sanasi |
| `description` | Tavsif |

**Hisoblanadigan ko'rsatkichlar:** kampaniyaga bog'langan mijozlar soni,
konversiya (won leads), byudjet bandligi (%).
**Ruxsatlar:** `campaigns.view`, `campaigns.create`, `campaigns.edit`, `campaigns.delete`.

---

### 3. 📈 Tahlil (Analytics)
Faqat o'qish uchun panel — kampaniyalar va mijozlar bo'yicha umumlashtirilgan
hisobot va grafiklar.

**Ko'rsatkichlar:**
- Jami mijozlar, jami kampaniyalar, faol kampaniyalar
- Umumiy byudjet va sarflangan summa
- Konversiya darajasi (won / jami leads)
- Mijozlar holati bo'yicha taqsimot (bar chart)
- Manba bo'yicha taqsimot
- Kanal bo'yicha kampaniya byudjeti
- Eng samarali kampaniyalar (top 5)

**Ruxsat:** `analytics.view`.

## Texnik amalga oshirish

### Migrationlar
- `create_campaigns_table`
- `create_leads_table` (FK: `campaign_id` → campaigns, `assigned_to` → users)

### Modellar
- `App\Models\Campaign` — `hasMany(Lead)`
- `App\Models\Lead` — `belongsTo(Campaign)`, `belongsTo(User, assigned_to)`
- `App\Models\User` — `hasMany(Lead, assigned_to)` (assignedLeads)

### Controllerlar
- `CampaignController` (resource, except show)
- `LeadController` (resource, except show)
- `AnalyticsController` (index)

### Viewlar
- `resources/views/campaigns/{index,create,edit}.blade.php`
- `resources/views/leads/{index,create,edit}.blade.php`
- `resources/views/analytics/index.blade.php`

### Routelar (auth middleware ichida)
```php
Route::get('/analytics', [AnalyticsController::class, 'index'])->name('analytics.index');
Route::resource('campaigns', CampaignController::class)->except(['show']);
Route::resource('leads', LeadController::class)->except(['show']);
```

### Ruxsatlar (RolePermissionSeeder)
- `superadmin` → barcha yangi ruxsatlar
- `manager` → `leads.*`, `campaigns.view`, `analytics.view`

### Navigatsiya
- Sidebarga "Marketing" bo'limi: Mijozlar, Kampaniyalar, Tahlil
- Dashboardga yangi statistika kartalari
